# Lab 01 – Azure Free Account & Cost Control

## Objective
Create an Azure Free account and configure cost control mechanisms to avoid unexpected charges.

## Tasks Performed
- Created Azure Free account
- Verified payment method ($1 USD authorization)
- Configured Budget in Cost Management
- Created cost alerts based on percentage of budget
- Reviewed Cost Management + Billing options

## Key Concepts Learned
- Azure Free provides limited credits and services
- Budgets do NOT stop resources, they only send alerts
- Alerts can be based on:
  - Actual cost
  - Forecasted cost
- Importance of deleting unused resources

## Real-World Value
Cost control is critical in cloud environments to prevent overspending and manage resources efficiently.
